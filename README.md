# 慈濟安平聯絡處

慈濟安平聯絡處社區網站，提供活動資訊、社區課程、社群動態及聯絡資訊。

**Website**: https://anping.tzuchi-tainan.cc

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Development

```sh
# Clone
git clone git@github.com:tzuchi-anping/anping-peace-hub.git
cd anping-peace-hub

# Install dependencies
npm install

# Start dev server
npm run dev
```

Dev server runs at `http://localhost:8080`

## Deployment

Push to `main` branch will automatically trigger GitHub Actions to build and deploy to GitHub Pages.

```sh
# Build
npm run build
```
